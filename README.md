## 投稿规范
### 1.标题

正确示范：C++：添加新功能

### 2.内容
一般分为两段，第一段为描述投稿的内容，放在哪个文件里，第二段为备注（例如标明该段代码的贡献者，在哪里标等），具体代码放附件最好带注释，不要定义一些使维护变难的变量名，不能为exe等可执行文件，不能放恶意程序。
